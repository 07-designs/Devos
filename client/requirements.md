## Packages
framer-motion | Complex animations for the cyberpunk interface
recharts | Visualizing developer statistics
react-markdown | Rendering AI analysis reports
date-fns | Formatting dates for activity logs
clsx | Conditional class names utility
tailwind-merge | Merging tailwind classes safely

## Notes
The application relies on Replit Auth for user management.
AI features use the backend /api/ai/analyze endpoint which connects to Gemini.
Images should be handled via CSS or Unsplash placeholders where appropriate.
Theme is strictly Cyberpunk/Hacker: dark mode, neon accents, monospace fonts.
